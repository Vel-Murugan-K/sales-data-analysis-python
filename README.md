# Sales Data Analysis

## Project Description
- This project analyzes sales data to calculate total sales revenue.
- It identifies top-performing and low-performing products.
- It evaluates category-wise and monthly sales performance.
- It generates a structured dataset suitable for business reporting and analysis.
- It processes raw sales input data containing order details, product information, and pricing.
- It validates and organizes input data to ensure accurate calculations and analysis.
- It prepares the data in a clean format for further reporting and decision-making.

## Tools Used
- Python
- Pandas
- Google Colab
- GitHub

## Input
- Raw sales data containing order ID, order date, product name, category, quantity, and price.
- Sales records collected over multiple months for trend analysis.

## Output
- week2_sales_analysis_output.csv
